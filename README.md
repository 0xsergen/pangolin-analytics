# Pangolin Analytics

## To Start Development

### Installing dependencies
```bash
yarn
```

### Running locally

#### Set an Deployment Environment
```bash
export DEPLOYMENT_NAME='hedera-mainnet'
```

#### Start App
```bash
yarn start
```
